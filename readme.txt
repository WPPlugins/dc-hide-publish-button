== Description ==
This plugin will come handy for author who often accidentally click publish button when what what they realy want is save only.
This plugin hide publish button for Post and Page which status is Draft. 
Publish button will shows when status change to Pending Review, or if you change Visibility state which will automatically change Post / Page status.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
